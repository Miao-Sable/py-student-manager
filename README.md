## 添加学生
1. 显示提示：打印“---添加学生---”。
2. 输入学号：获取用户输入的学号。
3. 检查学号是否重复：遍历现有的学生列表，如果发现某个学生的 id 与输入的学号相同，则提示“学号已存在”，直接结束函数（不再继续添加）。
4. 输入姓名：如果学号不重复，再提示用户输入姓名。
5. 输入三门成绩：分别输入数学、语文、英语成绩，并处理可能的非数字输入（用 try...except 捕获错误，若输入非数字则提示失败并结束函数）。
6. 构建学生字典：将学号、姓名、三门成绩存入字典。
7. 添加到列表：将字典添加到 students 列表中。
8. 提示成功：打印“添加成功”。

## 显示所有信息
1. 输入：一个学生字典列表。
2. 处理：判空 → 打印表头 → 遍历计算总分并打印。
3. 输出：格式化的学生成绩表格（打印到屏幕）。

## 删除学生（按学号）
1. 输入：要删除的学生学号
2. 处理：查找学号 -> 找到删除/没找到打印

## 修改学生成绩
1. 输入学号：提示用户输入要修改的学生的学号。 查找学生：遍历 students 列表，寻找 id 与输入匹配的学生。 如果找不到，打印“未找到该学号”并结束函数。 
2. 如果找到， 显示当前信息：打印该学生的全部当前数据，让用户知道原来的成绩。
3. 依次修改成绩（数学、语文、英语）： 对每一科，提示输入新成绩，并说明“直接回车表示不修改”。 
4. 获取用户输入。 如果输入不为空，尝试将输入转换为浮点数并赋值给对应的成绩字段。 如果输入为空，则跳过该科（保留原值）。
5. 异常处理：在转换成绩时，如果用户输入了非数字内容，会触发 ValueError，捕获后提示“成绩必须是数字”，并直接退出修改函数（不保存任何修改）。 完成修改：如果所有输入都合法（或跳过），最后打印“修改成功”并返回。
